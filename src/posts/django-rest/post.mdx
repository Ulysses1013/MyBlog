### 1. 仮想環境構築

---

<br />

はじめに**作業ディレクトリ**に移動して、**ターミナル**を開く。

```terminal
python3 -m venv venv
```

これで**仮想環境を構築**します。

次に**仮想環境に入ります。**

```terminal
venv\Scripts\activate // windowsの場合

source venv/bin/activate // macの場合
```

### 2. インストール

---

<br />

そして、**必要なもの**をインストールします。

```terminal
pip install django
pip install djangorestframework
```

他にも**色々便利なもの**がありますが、<br/>
**チュートリアル**なのでこれだけにします。

### 3. プロジェクト作成

---

<br />

**プロジェクト**を作成します。

```terminal
django-admin startproject プロジェクト名

django-admin startapp アプリ名
```

その後、**Settings.py**を開き、**rest_framework**を追加します。

```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'rest_framework', # ここ
]
```

これで完了です！

ここまでが**Django REST Framework を使い始めるまでの流れ**です。
