### favicon 設定方法

---

<br />

はじめにプラグインを**インストール**します。

`npm install gatsby-plugin-manifest`

次に gatsby-config.js に設定します。

```js
// in gatsby-config.js
module.exports = {
  plugins: [
    {
      resolve: `gatsby-plugin-manifest`,
      options: {
        name: `GatsbyJS`,
        short_name: `GatsbyJS`,
        start_url: `/`,
        background_color: `#f7f0eb`,
        theme_color: `#a2466c`,
        display: `standalone`,
        icon: `faviconのあるファイルのパス`, //ここ
      },
    },
  ],
}
```

これで**gatsby ブログに favicon**を設定できました。
